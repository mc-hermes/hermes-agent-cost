# Zaim's Knowledge Base Dashboard

A live knowledge base powered by [gbrain](https://github.com/nousresearch/gbrain), deployed as a static site.

## What this is

Every note, meeting, client detail, and project update in one searchable, connected dashboard. Forward an email, save a note, and the brain maps the connections for you.

## How to use

1. Open `index.html` in any browser — or visit the live URL
2. Click any note to read it, then follow the links
3. Use the graph to see how everything connects
4. Search for anything — people, projects, ideas
5. Write new notes directly from the dashboard

## For your clients and colleagues

Share the URL. They can browse your public knowledge, search for context, and see the visual map of how your work connects. No login, no install, no app.

## Built with

- [gbrain](https://github.com/nousresearch/gbrain) — local-first knowledge base
- GitHub Pages — zero-cost hosting
- Single HTML file — no frameworks, no build step
