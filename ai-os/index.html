AI OS Architecture

Karpathy-inspired · Knowledge-Kernel Design
Three-tier architecture inspired by Karpathy's Second Brain / LLM Wiki and the gbrain knowledge-graph kernel. Each tier inherits the layer below — the kernel is shared, the interfaces expand.

● Tier 1 — Individual

  • Single user, single agent profile
  • Personal knowledge base (private gbrain)
  • Messaging gateway (Telegram/Discord DM)
  • Personal memory & session persistence
  • Simple cron for reminders & briefings
  • Cost: ~$10–20/mo (VPS + API)

● Tier 2 — Department

  • 2–50 users, shared context
  • Kanban board for task orchestration
  • Role-gated access (viewer/editor/admin)
  • Shared knowledge base + project tracker
  • Delegation & profile workers
  • Cost: ~$30–100/mo (multi-agent VPS)

● Tier 3 — Business

  • Org-wide knowledge graph
  • Agent swarms with orchestration
  • Compliance audit trails
  • External system integration (API gateways)
  • Multi-tenant data isolation
  • Cost: ~$100–500/mo (scaled infra)
