gbrain SaaS — Single-Tenant Managed Architecture

Per-client LXC container • BYO OpenAI keys • Client-owned Google Drive • Operator-managed runtime

Data Privacy Model

  • Client owns Google Drive folder
  • OpenAI calls use client's own API key
  • PGLite per-client, isolated at filesystem level
  • Full data purge on cancellation
  • Dashboards behind auth (not public)

Monthly Subscription Stack

  • Tier: Starter (50 pages) / Pro (500) / Scale
  • Client pays: subscription + own OpenAI usage
  • Operator cost: VPS + LXC overhead (~$2-5/mo/client)
  • Billing: Stripe + usage metering
  • Self-service onboarding with OAuth flow

gbrain Runtime Constraints

  • PGLite = embedded WASM, not a remote server
  • Bun runtime required (no browser WASM port)
  • Single-file DB: no concurrent writers
  • CLI commands are synchronous / blocking
  • Job queue needed for long builds
